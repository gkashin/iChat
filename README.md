## Getting Started
* Initialize Podfile:
```
pod init
open Podfile
```
* Add pods to Podfile:
```
pod 'Firebase/Auth'
pod 'GoogleSignIn'
pod 'Firebase/Firestore'
pod 'Firebase/Storage'
pod 'SDWebImage', '~> 5.0'
pod 'MessageKit'
```
* Install pods:
```
pod install
open iChat.xcworkspace
```

## Enjoy!
